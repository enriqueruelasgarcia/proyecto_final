API PROYECTO FINAL

Proyecto Web de punto de venta, este debe contar con una API con seguridad mediante JWT La interfaz con HTML y CSS o bootstrap debe permitir que se identifique un usuario y tenga acceso al menú para dar de alta nuevos productos, clientes, vendedores y a su vez, pueda generar una factura, la factura se compone de varios productos, mismos que serán descontados de la cantidad de productos en la BD. Se podrán consultar facturas por clientes y ver el desglose de la información. Entregar el repositorio que se deberá componer de 2 carpetas principales /front y /api, dentro de la API, deberán subir un archivo script.sql con las instrucciones de creación de la BD y las tablas correspondientes.
